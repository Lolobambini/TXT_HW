БР-1  Отсутствует страница при переходе в раздел документации на сайте https://www.python.org
	Описание:
	      Предусловие: открыть браузер во весь экран
	Шаги:
	1. Открыть в браузере сайт www.python.org
	2. В меню под логотипом выбрать наведением мышки пункт «Documentation»
	3. Нажать кнопку «Python Docs»
	
  Ожидаемый результат: Осуществлен переход на страницу Python 3.9.6 documentation
	
  Полученый результат: Ошибка 404, «Страница не найдена»
	
  Среда: Windows 7, Google Chrome Версия 92.0.4515.159 (Официальная сборка), (64 бит)
